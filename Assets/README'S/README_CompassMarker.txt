Making an object show on the compass

Step 1. Add the CompassMarker component to the object that you wish to show up on the compass.

Step 2. Make a new scriptable object for the compass marker.
 Assets->ScriptableObjects->CompassMarker->Right click-> Create -> CompassMarker

Step 3. Assign the sprite you wish to represent this object in the "Compass Marker Sprite" slot in the inspector.

Step 4. Assign the newly created scriptable object in the CompassMarker component on your object.
